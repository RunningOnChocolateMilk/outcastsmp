Frequently Asked Questions

General Questions

What version of Minecraft is required?

Outcast SMP runs on Minecraft Java Edition version 1.21.7. Make sure you have this exact version installed.

Is the server really anarchy?

Yes! Outcast SMP is an anarchy server with fair play rules to prevent exploits and ensure balanced gameplay. Outside of these rules and spawn protection, players can grief, PvP, raid, and play freely. Check the Server Rules page for complete details.

Do I need to install mods?

No! Outcast SMP uses server-side mods only. You do not need to install any mods on your client. Just connect with vanilla Minecraft Java Edition 1.21.7 and all server features will work automatically.

Can I use TLauncher?

Yes! The server supports both official Minecraft Java Edition and TLauncher players. All server features work regardless of which launcher you use, as everything is handled server-side.

How do I get the server IP?

The server IP is 15.235.159.106:25600. You can also find connection details on our Discord server.

Gameplay Questions

Will my base get griefed?

Almost certainly, yes. That's part of the anarchy experience. Hide your base well and build far from spawn to reduce the chances.

Can I team up with other players?

Absolutely! Forming groups is a common survival strategy, but remember - trust is rare in anarchy. Anyone could betray you.

Are hacks/cheats allowed?

This depends on server policy. Some anarchy servers allow client-side modifications, others don't. Check with admins or on Discord for the specific rules.

What happens when I die?

You'll respawn at spawn (or your bed if you have one), and you'll drop all your items where you died. Someone might take them before you can retrieve them.

Technical Questions

What are the server specs?

Check the "Server Information" section for technical details about the server hardware and performance.

How much RAM do I need to run the modpack?

We recommend at least 4GB of RAM allocated to Minecraft, though more is always better for performance.

The server is lagging, what should I do?

Lag can happen on any server. Try reducing your render distance, turning down graphics settings, or allocating more RAM to Minecraft.
