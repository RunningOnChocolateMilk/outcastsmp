Performance

We strive to maintain good server performance, but lag can occur due to:

  • Large player count
  • Resource-intensive builds
  • Massive farms and redstone contraptions
  • World size and exploration

We monitor performance and optimize when possible, but in anarchy, players often push the limits!

Backups

Regular backups are maintained for server stability. However:

  • Individual player griefing will not be reverted
  • Backups are for server crashes, not player actions
  • Deaths and item loss are permanent

Updates and Maintenance

  • Server updates will be announced on Discord
  • Scheduled maintenance will be posted in advance
  • Emergency restarts may occur with minimal notice
  • Join Discord to stay informed about server status
