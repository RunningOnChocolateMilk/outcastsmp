Getting Started

Prerequisites

To join Outcast SMP, you'll need:

  • Minecraft Java Edition 1.21.7 (official or TLauncher)
  • No mods required! All mods are handled server-side - just connect with vanilla Minecraft
  • A Discord account (join our server for updates and community)

Note: The server supports both official Minecraft Java Edition and TLauncher players. No client-side mods are required regardless of which launcher you use!

Joining the Server

  1. Launch Minecraft Java Edition 1.21.7 using your preferred launcher (official launcher, TLauncher, etc.) - no mods needed!
  2. Add the server to your server list:
    • Server IP: 15.235.159.106:25600
  3. Connect and start your anarchy adventure!

First Steps

When you first join, remember:

  • Read the Server Rules - there are fair play guidelines
  • This is an anarchy server - players can grief, PvP, and raid (outside spawn)
  • Trust no one, but make alliances if you dare
  • Your builds are not safe unless well hidden (spawn is protected)
  • Join our Discord for server updates and community events
